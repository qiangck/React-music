# React-music

## 概述

项目为基于`React`开发的播放器，此为v1.0版本，问题还很很多，希望能有朋友能提出建议修改。

## 使用

#### 方法一：
使用`npm`进行安装
```
npm install
npm run start
```

#### 方法二：
使用`yarn`进行安装
```
yarn
yarn run start
```

## 手机预览

http://localhost:3000/
